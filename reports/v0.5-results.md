## False Positive / False Negative Analysis

Regex baseline:

- 0 false positives.
- 63 false negatives.
- Most misses are paraphrases, cross-file forms, multi-label rollback labels,
  and non-canonical terminal/release wording. This is expected for a transparent
  phrase baseline.

CtxGov doctor adapter:

- 0 false positives.
- 0 false negatives on the deterministic v0.5 scaffold.
- Evidence-span token-F1 is 1.0000 because the doctor now emits exact native
  spans for the scaffold's public phrases and paraphrases.

## What The 1.0000 Means

The CtxGov doctor result means the main repo doctor, companion adapter,
materialized local workspaces, multi-label scorer, and evidence-span extraction
are internally consistent on this deterministic mutation scaffold.

It does not mean CtxGov has a general benchmark result, security coverage,
provider compatibility, real-world prevalence estimate, or independent
reviewer-validated performance.

## Remaining Research Gap

The next high-value research step is not more deterministic mutation coverage.
It is adding trace-derived reviewed data without making human review a blocker
for local progress:

- keep local synthetic and mutation tests as the default regression gate
- collect trace-derived cases into an intake split
- label with a local rubric first
- later add independent review and adjudication as a quality upgrade
- report disagreements separately from scaffold metrics
