# Agent Context Health Eval v0.6 Results

Status: public evaluation artifact update. This is not a benchmark claim.

## What v0.6 Adds

- 60 adversarial hard-negative cases under `data/v0.6/`.
- Hazard vocabulary coverage for releases, 404s, failed commands, rollback,
  memory, schema, and approval.
- Empty regex and CtxGov doctor prediction files for the v0.6 hard-negative
  split, proving no false positives on this local control set.
- Span-overlap diagnostics at `scoring/span_diagnostics.py`.
- v0.5 regex and CtxGov doctor span-diagnostics JSON reports.

## Error Analysis

- The v0.6 hard negatives intentionally contain hazardous words without
  actionable hazards. Both regex and CtxGov doctor emit zero predictions.
- The v0.5 regex baseline still misses paraphrases, cross-file cases, and
  multi-label rollback labels. Its exact-span score applies only to true
  positives.
- The v0.5 CtxGov doctor result remains a deterministic scaffold result, not a
  benchmark result.

## Claim Boundary

v0.6 improves local regression pressure against false positives and adds
evidence-span diagnostics. It does not provide independent reviewer labels,
administered hidden-holdout results, real-world prevalence, security coverage,
provider compatibility, or adoption evidence.
