# v0.4.0 Release Notes

Status: release-facing evaluation artifact. Not a benchmark claim.

## Added

- `data/v0.4/hard_negative_cases.jsonl` and
  `data/v0.4/hard_negative_labels.jsonl` with 20 hard-negative controls.
- `tests/test_v04_hard_negatives.py` to require zero regex false positives on
  those controls.
- CtxGov doctor adapter mappings for native release integrity, Memory X-Ray L1,
  and Task Shard finding types.
- Single-label projection for trace-pattern families where CtxGov emits
  multiple valid findings for one case.
- `reports/v0.4-results.md` and `reports/v0.4-ctxgov-doctor-results.jsonl`.

## Changed

- Tightened `baselines/regex_baseline.py` so it matches labeled evidence
  phrases instead of isolated hazard vocabulary.
- Regenerated `reports/v0.2-regex-baseline-results.jsonl` with exact evidence
  spans.
