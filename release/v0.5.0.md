# Agent Context Health Eval v0.5.0 Release Notes

Status: prepared GitHub source release draft. This file alone does not publish
the release.

v0.5.0 adds deterministic mutation and multi-label evaluation coverage for the
Agent Context Health artifact.

## Added

- `data/v0.5/mutation_cases.jsonl`: 160 deterministic mutation cases.
- `data/v0.5/mutation_labels.jsonl`: 206 labels with exact evidence spans.
- `data/v0.5/mutation_manifest.json`: case, label, clean-control, and
  mutation-kind counts.
- `scripts/generate_v05_mutation_data.py`: reproducible local data generator.
- `scoring/score_multilabel.py`: finding-level and case-level multi-label
  scoring.
- `baselines/regex_baseline.py --multi-label`: transparent regex baseline that
  can emit more than one prediction per case.
- `ctxgov_adapter/run_ctxgov.py --projection none`: native CtxGov doctor
  adapter path without single-label projection.
- `reports/v0.5-results.md`: local v0.5 scaffold metrics and limitation notes.

## Claim Boundary

This release is a deterministic local scaffold for schema, scorer, adapter, and
doctor regression testing. It is not a public benchmark result, security
guarantee, provider compatibility claim, agent-safety claim, hosted demo claim,
or adoption claim.
