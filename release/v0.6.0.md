# Agent Context Health Eval v0.6.0 Release Notes

Status: prepared GitHub source release draft. This file alone does not publish
the release.

v0.6.0 adds adversarial hard-negative controls and evidence-span diagnostics to
the Agent Context Health evaluation artifact.

## Added

- `data/v0.6/adversarial_hard_negative_cases.jsonl`: 60 clean controls with
  hazardous vocabulary in repaired, scoped, or negated context.
- `data/v0.6/adversarial_hard_negative_labels.jsonl`: `none` labels for all
  v0.6 hard-negative cases.
- `scripts/generate_v06_adversarial_hard_negatives.py`: reproducible local data
  generator.
- `reports/v0.6-regex-hard-negative-results.jsonl`: empty regex prediction
  output for v0.6 hard negatives.
- `reports/v0.6-ctxgov-doctor-hard-negative-results.jsonl`: empty CtxGov doctor
  prediction output for v0.6 hard negatives.
- `scoring/span_diagnostics.py`: evidence-span overlap distribution and
  near-miss diagnostics.
- `reports/v0.5-regex-span-diagnostics.json` and
  `reports/v0.5-ctxgov-doctor-span-diagnostics.json`.
- `reports/v0.6-results.md`: v0.6 local results and claim boundary.

span diagnostics. It is not a public benchmark result, security guarantee,
provider compatibility claim, agent-safety claim, hosted demo claim, or adoption
claim.
